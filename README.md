<h1>Metamorphosis</h1>

Metamorphosis is a ThreeJS powered website that features a rhythm game esque mini-game. The artwork depicts a digital clone at the end of its life, asking the user if it is of sound mind to be given the power of attorney over their orginal organic self. The messaging and themes of the work are in parallel to an essay that can be found here:
[Computational Ethics Eluned Owen (3).pdf](https://github.com/user-attachments/files/25825060/Computational.Ethics.Eluned.Owen.3.pdf).
